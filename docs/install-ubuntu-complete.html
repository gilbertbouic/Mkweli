Before You Start
This guide assumes you have never used the Terminal before. We'll explain everything step by step. The installation takes about 10-15 minutes.

1. What is Terminal? (and how to open it)

The Terminal (sometimes called "command line" or "shell") is a program where you type commands to your computer instead of clicking buttons. Think of it like having a text conversation with your computer.

üí°
Don't worry!
The Terminal might look intimidating at first, but for this installation you just need to copy and paste one command. We'll show you exactly how!

How to Open Terminal on Ubuntu

Method 1: Keyboard Shortcut (Easiest!)

Press these three keys at the same time:

Ctrl + Alt + T

A window with a dark background will open - this is the Terminal!

Method 2: Using the Activities Menu

1

Click "Activities" in the top-left corner

Look at the very top-left corner of your screen. Click on the word Activities.

üñºÔ∏è

"Activities" is in the top-left corner of your Ubuntu desktop

2

Type "Terminal"

With the Activities view open, just start typing the word Terminal. You don't need to click anywhere - just type!

3

Click the Terminal icon

You'll see a black/dark icon with a white rectangle inside. Click on it to open Terminal.

üñºÔ∏è

The Terminal icon looks like a black box with a white cursor inside

‚úÖ
What you should see
A dark window will open with some text and a blinking cursor. It might show something like: yourname@yourcomputer:~$ - this means the Terminal is ready for you to type!

2. One-Command Installation

The easiest way to install Mkweli AML is with our automatic installer. Just copy and paste one command!

üöÄ Copy and paste this entire command:

curl -fsSL https://raw.githubusercontent.com/gilbertbouic/Mkweli/main/scripts/install-ubuntu.sh | bash

üí° This command downloads and runs our installation script automatically.

üîí
Security Note
If you prefer to inspect the script before running it, you can:
  1. Download: curl -fsSL https://raw.githubusercontent.com/gilbertbouic/Mkweli/main/scripts/install-ubuntu.sh -o install-ubuntu.sh
  2. Review: less install-ubuntu.sh
  3. Run: bash install-ubuntu.sh

How to copy and paste in Terminal

1

Copy the command above

Use your mouse to highlight the command in the orange box above. Then:

  • Right-click and select "Copy", OR
  • Press Ctrl + C
2

Click inside the Terminal window

Click anywhere inside the Terminal window to make sure it's active.

3

Paste the command

Important: In Terminal, the paste shortcut is different!

  • Right-click and select "Paste", OR
  • Press Ctrl + Shift + V

You should see the command appear in the Terminal.

4

Press Enter to run

Press the Enter key to run the command.

5

Enter your password when asked

The installer needs permission to install software. When it asks for your password:

  • Type your Ubuntu login password
  • Note: You won't see the letters as you type - this is normal for security!
  • Press Enter when done
6

Wait for installation to complete

The installer will:

  • Install Docker (if not already installed)
  • Download Mkweli AML
  • Set everything up automatically

This takes about 5-10 minutes. You'll see lots of text scrolling - that's normal!

‚ö†Ô∏è
Stay connected to the internet!
The installation downloads about 500MB of data. Make sure you have a stable internet connection.

Alternative: Manual Installation

If the one-liner doesn't work, you can run these commands one at a time:

Copy each line # Install Docker sudo apt update sudo apt install -y docker.io docker-compose # Add yourself to docker group sudo usermod -aG docker $USER # Download Mkweli git clone https://github.com/gilbertbouic/Mkweli.git cd Mkweli # Create config file cp .env.example .env # Start Mkweli docker-compose up -d
‚ÑπÔ∏è
After manual installation
You need to log out and log back in for Docker permissions to work. Click the power icon in the top-right corner and select "Log Out".

3. Verify Installation Worked

Let's make sure everything installed correctly.

1

Check Docker is running

Open Terminal and type this command, then press Enter:

You should see a table with columns like "CONTAINER ID", "IMAGE", etc. If you see "mkweli" in the list, Mkweli is running!

2

Open Mkweli in your browser

Open your web browser (Firefox is usually pre-installed on Ubuntu).

In the address bar, type: http://localhost:8000

Press Enter.

3

You should see the login page

If you see the Mkweli AML login page, congratulations - installation was successful!

6. Daily Workflow

Here's how to use Mkweli AML on a typical day:

üåÖ Morning: Start the App

Mkweli usually stays running, but if you need to start it:

  1. Open Terminal (Ctrl + Alt + T)
  2. Type: cd ~/Mkweli && docker-compose up -d
  3. Press Enter
  4. Open browser and go to http://localhost:8000

‚òÄÔ∏è During the Day: Use the App

  1. Open your browser to http://localhost:8000
  2. Log in with your password
  3. Use "Quick Check" for single names
  4. Use "Upload List" for many names (CSV/Excel)
  5. Download and save your reports

üåô Evening: Stop the App (optional)

You can leave Mkweli running, but if you want to stop it:

  1. Open Terminal
  2. Type: cd ~/Mkweli && docker-compose down
  3. Press Enter
üí°
Tip: Bookmark the page!
Press Ctrl + D in your browser to bookmark http://localhost:8000 for quick access.

7. Next-Day Restart Procedure

If Mkweli is still running (common)

Usually Mkweli stays running even after you restart your computer. Just:

  1. Open your browser
  2. Go to http://localhost:8000
  3. Log in and start working!

If Mkweli is not running

1

Open Terminal

Press Ctrl + Alt + T

2

Start Mkweli

Type this command and press Enter:

cd ~/Mkweli && docker-compose up -d
3

Wait about 30 seconds

Then open your browser to http://localhost:8000

8. Troubleshooting for Non-Technical Users

Having problems? Here are solutions to the most common issues:

‚ùå "Permission denied" when running Docker commands

Solution:
  1. Run this command in Terminal:
  2. sudo usermod -aG docker $USER
  3. Enter your password when asked
  4. Log out and log back in (click power icon ‚Üí Log Out)
  5. Try again

‚ùå "Cannot connect to localhost:8000" in browser

Solution:
  1. Open Terminal and run: docker ps
  2. If no containers are listed, run: cd ~/Mkweli && docker-compose up -d
  3. Wait 30 seconds and try again
  4. Try using http://127.0.0.1:8000 instead

‚ùå Installation failed or command not found

Solution:
  1. Make sure you have internet connection
  2. Try running: sudo apt update && sudo apt install -y curl git
  3. Then try the installation command again

‚ùå I forgot my password

Solution:

Contact support at gilbert@mkweli.tech for help resetting your password.

‚ùå "docker-compose: command not found"

Solution:

Run this command to install docker-compose:

sudo apt install docker-compose -y

‚ùå Computer is slow when running Mkweli

Solution:
  • Close other applications you're not using
  • Make sure you have at least 4GB RAM (8GB recommended)
  • When not using Mkweli, stop it: cd ~/Mkweli && docker-compose down
