üåÖ Morning: Starting Your Day

‚òÄÔ∏è How to Start Mkweli AML

Step 1: Turn on your computer and log in

Wait for Windows to fully load. You'll see your desktop with icons.

Step 2: Wait for Docker to start

Look at the bottom-right corner of your screen (near the clock). Wait until you see a whale icon üê≥. If the whale is moving/animating, wait until it stops - this means Docker is ready.

Don't see the whale? Click Start, type "Docker Desktop", and click to open it.

Step 3: Start Mkweli

  1. Double-click the Mkweli AML shortcut on your desktop
  2. A menu window will appear
  3. Type 1 and press Enter (or click "Start Application")

Step 4: Wait for the browser to open

Your web browser will open automatically in about 30 seconds. If it doesn't, open your browser and go to: http://localhost:8000

Step 5: Log in

Enter your username and password, then click Login. You're ready to work!

Step 2: Check if Mkweli is already running

Open your browser and go to: http://localhost:8000

If you see the login page, skip to Step 5! If not, continue to Step 3.

Step 3: Open Terminal

Press Ctrl + Alt + T to open Terminal.

Step 4: Start Mkweli

Type this command and press Enter:

cd ~/Mkweli && docker-compose up -d

Wait about 30 seconds for it to start.

Step 5: Open Mkweli

Open your browser and go to: http://localhost:8000

Log in with your username and password. You're ready to work!

‚òÄÔ∏è Afternoon: Using Mkweli

üíº Basic Features

Quick Check: Screen a Single Name

  1. From the dashboard, click "Quick Check"
  2. Enter the name you want to screen
  3. Optionally add date of birth and nationality
  4. Click "Check"
  5. Review the results - you'll see if there are any matches

Batch Screening: Check Many Names at Once

  1. From the dashboard, click "Upload List"
  2. Click "Choose File"
  3. Select your CSV or Excel file with names
  4. Click "Upload and Screen"
  5. Wait for processing (this may take a few minutes for large files)
  6. Download the PDF report when ready
üìÑ
File Format
Your file should have columns for: Name, Date of Birth (optional), Nationality (optional). The first row should be headers.

Understanding Results

  • No Matches Found - The name is not on any sanctions list (good!)
  • Potential Match - A similar name was found - review carefully
  • Exact Match - The exact name appears on a sanctions list - verify immediately

Saving Reports

  1. After screening, click "Download Report"
  2. Save the PDF file to a safe location
  3. Reports include a unique hash for verification purposes

Tip: Leave Mkweli Running!
You can leave Mkweli running all day. Just close the browser tab when you're not using it - the application will still be running and ready when you come back.

üåô Evening: Closing Down

üåú How to Properly Close Mkweli

Option A: Leave it running (Recommended)

You can simply close your browser and leave Mkweli running. It will be ready instantly tomorrow morning!

Option B: Stop Mkweli completely

  1. Save any work you haven't saved
  2. Double-click the Mkweli AML shortcut on your desktop
  3. Type 2 and press Enter (or click "Stop Application")
  4. Wait for the confirmation message
  5. You can now shut down your computer

Option A: Leave it running (Recommended)

You can simply close your browser and leave Mkweli running. It will be ready instantly tomorrow morning!

Option B: Stop Mkweli completely

  1. Save any work you haven't saved
  2. Open Terminal (Ctrl + Alt + T)
  3. Type this command and press Enter:
cd ~/Mkweli && docker-compose down

Wait for the confirmation message, then you can shut down your computer.

‚ö†Ô∏è Important: What NOT to Do

‚ùå DON'T Do These Things:

  • Don't force-close - Never use Task Manager (Windows) or "kill" command (Ubuntu) to close Mkweli. This can corrupt your data.
  • Don't delete files - Never delete files from the Mkweli folder unless you know exactly what you're doing.
  • Don't uninstall Docker - Mkweli needs Docker to run. If you uninstall Docker, Mkweli will stop working.
  • Don't share your password - Your login credentials are for you only.
  • Don't ignore the "Update Data" warning - When Mkweli says sanctions data is old, update it as soon as possible.
  • Don't run multiple copies - Only run one instance of Mkweli at a time.

‚úÖ DO These Things:

  • Do save reports - Always download and save your screening reports to a safe location.
  • Do use the proper stop command - When stopping Mkweli, use the launcher menu or the correct terminal command.
  • Do keep Docker running - Make sure Docker Desktop (Windows) is running before starting Mkweli.
  • Do update sanctions data regularly - Check for updates at least every 2 weeks.
  • Do log out when away - If stepping away from your computer, log out of Mkweli for security.
  • Do backup your data - Periodically backup the Mkweli folder to an external drive or cloud storage.
