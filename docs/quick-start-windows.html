üéÆ Using the Launcher

The desktop shortcut opens a simple menu with these options:

Option What It Does
1. Start Application Starts Mkweli and opens your browser
2. Stop Application Stops Mkweli (saves resources)
3. Restart Application Restarts Mkweli (fixes most issues)
4. Open Dashboard Opens the web interface in your browser
5. View Logs Shows recent activity (for troubleshooting)
6. Check Status Shows if everything is running correctly

üîß Common Issues

"Docker Desktop is not running"

Solution: Open Docker Desktop from the Start Menu and wait for the whale icon to stop animating (about 30 seconds). Then try again.

"The application won't start"

Solution: Make sure Docker is running, wait 2-3 minutes (first start is slower), then try "Restart Application" from the launcher.

"I can't access localhost:8000"

Solution: Check if the application is running using "Check Status". Try typing http://127.0.0.1:8000 instead.

"My computer is slow"

Solution: Close other applications. Mkweli works best with 8 GB+ RAM. Stop the application when not in use.
