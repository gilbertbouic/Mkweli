1. What is Terminal? (and how to open it)

The Terminal (also called "Command Prompt" or "PowerShell" on Windows) is a program that lets you type commands to your computer instead of clicking buttons. Think of it like having a conversation with your computer using text.

üí°
Good News!
For Mkweli AML, you usually don't need to use the Terminal! The installer and desktop shortcuts do everything for you. But it's good to know where it is, just in case.

How to Open Terminal on Windows

1

Click the Start button

Look at the bottom-left corner of your screen. You'll see the Windows logo (it looks like four squares). Click it.

üñºÔ∏è

The Start button is at the bottom-left corner of your screen

2

Type "Terminal" or "cmd"

With the Start menu open, just start typing the word Terminal or cmd. You don't need to click anywhere first - just type. Windows will search as you type.

3

Click to open

You'll see "Terminal" or "Command Prompt" appear in the search results. Click on it to open.

üñºÔ∏è

A black or blue window will open - this is the Terminal

Keyboard Shortcut
You can also press Windows + R, type cmd, and press Enter to open the Terminal quickly.

1

Find the file you just downloaded (MkweliAML-Setup-x.x.x.exe) and double-click on it.

2

The installer will open. Read the welcome message and click Next.

4

Accept the license agreement

5

The installer will suggest where to put the program (usually C:\Program Files\Mkweli AML). You can leave this as-is and click Next.

6

Click the Install button. The installer will copy files to your computer. This takes 1-2 minutes.

You'll see a progress bar showing how much is done.

7

When finished, click Finish. You now have Mkweli AML installed!

Check if Docker is installed

1

9. Troubleshooting for Non-Technical Users

Having problems? Here are solutions to the most common issues:

‚ùå "Docker is not running" error

Solution:
  1. Look at the bottom-right corner of your screen for the whale icon üê≥
  2. If you don't see it, click Start and type "Docker Desktop"
  3. Click to open Docker Desktop
  4. Wait 1-2 minutes for the whale to stop moving
  5. Try starting Mkweli again

‚ùå The app won't start or takes forever

Solution:
  1. The first start can take 5 minutes - be patient!
  2. Make sure Docker is running (whale icon in taskbar)
  3. Try selecting "Restart Application" from the launcher menu
  4. If still not working, restart your computer and try again

‚ùå "Cannot connect to localhost:8000" in browser

Solution:
  1. Wait 30 more seconds - the app might still be starting
  2. Try typing http://127.0.0.1:8000 instead
  3. Make sure you selected "Start Application" first
  4. Try selecting "Check Status" from the launcher to see if the app is running

‚ùå I forgot my password

Solution:

Unfortunately, you may need to reinstall the application if you forgot your password. Contact support at gilbert@mkweli.tech for help.

‚ùå My computer is very slow when running Mkweli

Solution:
  1. Close other programs you're not using
  2. Make sure your computer has at least 4GB of RAM (8GB recommended)
  3. When not using Mkweli, select "Stop Application" to free up resources
  4. Consider closing Docker Desktop when you're done for the day

‚ùå Browser shows a blank page

Solution:
  1. Press F5 to refresh the page
  2. Clear your browser's cache (press Ctrl + Shift + Delete)
  3. Try a different browser (Edge, Chrome, Firefox)
  4. Restart the application using the launcher

‚ùå "Permission denied" or "Access denied" error

Solution:
  1. Right-click on the Mkweli AML shortcut
  2. Select "Run as administrator"
  3. Click "Yes" when Windows asks for permission
