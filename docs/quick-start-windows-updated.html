‚ùì Common Questions

What is Docker and why do I need it?

Docker is a program that runs Mkweli in a safe, isolated environment. Think of it like a special container that keeps Mkweli separate from other programs on your computer. It's free and required for Mkweli to work.

Can I use Mkweli without internet?

Yes! After the initial installation, Mkweli works completely offline. You only need internet to update the sanctions data files.

How do I know if it's working?

When you see the login page at http://localhost:8000 in your browser, everything is working. You can also use "Check Status" from the launcher menu.

What if my computer is slow?

Mkweli needs at least 4GB of RAM to run well. If your computer is slow, try closing other programs while using Mkweli, or use "Stop Application" when you're done.

How do I update the sanctions data?

The application will remind you when data is older than 14 days. Follow the prompts in the app, or see our detailed guide for manual updates.

What if something goes wrong?

Try "Restart Application" from the launcher menu - this fixes most issues. For persistent problems, check the Setup Verification Checklist.
