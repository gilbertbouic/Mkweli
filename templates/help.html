Help & Documentation

Welcome to MkweliAML Help. This open-source tool for low-budget users relies on free official sanctions sources (UN, OFAC, UK, EU, Canada). Data is auto-fetched on startup and stored locally in the database—no cloud dependency, reducing manipulation risks. Manual uploads are available as fallback.

  • Dashboard: Overview of clients, flagged cases, and activity. Lists show update dates from local DB.
  • Client Management: Add clients, check against sanctions (fuzzy match >80%), delete. Risk scores update automatically.
  • Sanctions Lists: Auto-updates from free sources. Click "Refresh Lists" for manual trigger. View counts and dates. If format changes, errors are logged in logs/app.log—use manual import.
  • Reports: Generate PDF/HTML with client details, matches, lists used, and SHA-256 hash for integrity.
  • Settings: Customize organization name for reports.
  • Offline Use: Assumes internet for initial fetches; retains local data thereafter. Disable auto-fetch in app.py for fully offline.
  • Troubleshooting: Check logs/app.log for errors. Duplicates are prevented via checks; DB remains lean.
