Welcome to MkweliAML

MkweliAML is an open-source sanctions screening tool designed for KYC/AML compliance. It uses manually downloaded sanctions data (UN, OFAC, UK, EU) and operates 100% offline after initial setup.

üîÑ Updating Sanctions Lists (Every 14 Days)
  1. Download fresh XML files from official sources (see Sanctions Lists page)
  2. Rename files exactly as specified:
    • un_consolidated.xml
    • uk_consolidated.xml
    • ofac_consolidated.xml
    • eu_consolidated.xml
  3. Replace old files in data/ folder
  4. Click "Reload Sanctions" on Dashboard or Sanctions Lists page

The system automatically detects changed files and rebuilds the database.

üîç How Matching Works

MkweliAML uses a 4-layer matching system:

  1. Exact Match (100%) - Direct name comparison
  2. Token Match (85-99%) - Word overlap comparison
  3. Phonetic Match (75-84%) - Sound-alike and abbreviations (JSC, LLC, Ltd)
  4. Fuzzy Match (70-74%) - Partial string matching

Default threshold: 70%. Names scoring above this appear as potential matches.
